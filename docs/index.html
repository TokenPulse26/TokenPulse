🔒 Local-First

All data stays on your machine in SQLite. No cloud service, no telemetry, no account required. Your API keys pass through — never stored.

☁️ + 🏠 Hybrid Tracking

Cloud APIs and local models in one dashboard. See what Claude, GPT, Ollama, and others are doing side by side.

📊 Real Usage Data

Token counts, cost estimates, latency, streaming vs non-streaming, model breakdowns, and 30-day activity trends — from actual request interception, not billing polls.

💰 Budget Alerts

Set spending limits per project or provider. Get notified before you blow past your budget. Track forecast vs actual spend.

🔍 Context Audit

Heuristic analysis of your AI usage patterns — catch prompt waste, model mismatches, and cache underuse.

⚡ Rust Proxy

Built on Axum. Fast, low overhead. Handles streaming SSE, extracts usage metadata on the fly, and forwards everything transparently.

Quick Start

# Clone and build
git clone https://github.com/TokenPulse26/TokenPulse.git
cd TokenPulse/src-tauri && cargo build --release

# Start proxy + dashboard
./target/release/tokenpulse &
python3 ../web-dashboard.py &

# Point your tool at the right route
OpenAI-compatible:  http://localhost:4100
Anthropic:         http://localhost:4100/anthropic
Ollama:            http://localhost:4100/ollama

# Open dashboard
open http://127.0.0.1:4200
