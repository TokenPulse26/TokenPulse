Three steps. Five minutes.

Best current path: technical testers on macOS Apple Silicon.

# Install TokenPulse
curl -fsSL https://tokenpulse.to/install.sh | bash

# Point your tool or agent at TokenPulse routes on port 4100
OpenAI-compatible:  http://localhost:4100
Anthropic:         http://localhost:4100/anthropic
Ollama:            http://localhost:4100/ollama

# Agent-install flow: point the human to the local dashboard
export TOKENPULSE_DASHBOARD_URL=http://127.0.0.1:4200

# Open dashboard
open http://127.0.0.1:4200

Current limitations

The honest version, so you know what you're signing up for.

  • LM Studio support is lower confidence than Ollama.
  • Pricing data may be stale for the newest model releases.
  • Best fit today: technical users comfortable with a local proxy and dashboard.
